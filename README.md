<H>ГЕНЕРАЦИЯ СИНТЕТИЧЕСКИХ ТРАНЗАКЦИОННЫХ ДАННЫХ</H>
<p align=justify> Разработка метода генерации синтетических табличных данных, включающих временные метки, который сохраняет свойства ряда реальной последовательности, обеспечивает сохранение конфиденциальности и превосходит существующие методы по качеству и производительности.</p>   

<p align=justify> Синтетические данные позволяют заменить реальные или обогатить их для: повышения доступности валидных данных для задач аналитики и машинного обучения; повышения разнообразия исторических данных; сохранения конфиденциальности данных; создания сценариев для моделирования процессов. Для этого синтетические данные по статистическим закономерностям должны быть идентичны реальным, сохранять корреляционные зависимости между соответствующими полями внутри таблиц, исключать возможность восстановления реальной конфиденциальной информации по сгенерированным значениям. </p>

<p align=justify> В работе поставленная цель достигается решением следующих задач:</p> 
<UL>
<LI>модификация GAN модели на случай двух дискриминаторов и супервизора; </LI>
<LI>использование условного вектора, кодирующего категории и время транзакций;  </LI>
<LI>генерация времени на основе пуассоновского процесса с использованием технологий глубокого обучения;  </LI>
<LI>применение дифференциально-приватных алгоритмов в автокодировщиках при кодировании данных и в методе генерации транзакций; </LI>
<LI>разработка модуля управления сценариями по категориям транзакций. </LI> 
</UL>
<H2></H2>Данные<\H2>
<p align=justify>Набор 1:  транзакции оnline ритейла (25900 записей, 16 колонок) - содержит информацию о транзакциях, совершённых покупателями через онлайн-платформу. В набор данных входят данные о приобретённых товарах, количестве каждого товара, дате и времени каждой транзакции, цене каждого товара, уникальном идентификаторе каждого покупателя, совершившего покупку, и стране, в которой находится каждый покупатель.  https://www.kaggle.com/datasets/abhishekrp1517/online-retail-transactions-dataset  </p>

<p align=justify>Набор 2: транзакции ритейла (1000000 записей, 13 колонок)- синтетический набор данных, созданный для моделирования потребительской корзины, предоставляя информацию о покупательском поведении клиентов и работе магазина. Столбцы содержат информацию о транзакциях, клиентах, товарах и покупательском поведении. https://www.kaggle.com/datasets/prasad22/retail-transactions-dataset </p>

<p align=justify>Набор 3: финансовые операции с разметкой мошенничества (5000000 записей, 18 колонок) - содержит 5 миллионов искусственно сгенерированных финансовых транзакций, предназначенных для имитации реального поведения в целях исследований по выявлению мошенничества и приложений машинного обучения. https://www.kaggle.com/datasets/aryan208/financial-transactions-dataset-for-fraud-detection. </p>

<p align=justify>Набор 4: транзакционные данные (2554758 записей,  12 колонок) - комплексный набор финансовых данных, объединяющий записи о транзакциях, информацию о клиентах и данные карт банковского учреждения, охватывающие десятилетие 2010-х годов. https://www.kaggle.com/datasets/computingvictor/transactions-fraud-datasets </p>

<p align=justify>Набор 5: транзакционные данные (7477306 записей, 24 колонки)  -  набор данных созданый на основе данных реальных транзакций, сгенерирован искусственно, чтобы избежать проблем с конфиденциальностью. Он включает ключевые характеристики, играющие решающую роль в выявлении мошенничества, такие как суммы транзакций, типы устройств, географическое положение, валюта, тип карты и метка «мошенничество», указывающая на подозрительность транзакции.https://www.kaggle.com/datasets/ismetsemedov/transactions </p>
